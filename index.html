Investigando a adoção de expressões lambda em Java

A versão mais recente do Java trouxe novas implementações, incluindo a adoção de expressões lambda que tornam o código mais con- ciso e mais fácil de implementar. Além disso, a evolução da linguagem está fortemente ligada à evolução do software. Está pesquisa tem o objetivo de investigar a nova implementação de expressões lambda na linguagem Java.

Palavras-chave: lambda-calculus, Web Mining, Mining in software repositories, Software repositories, Java 8, lambda expressions.

PLANEJAMENTO

A tabela abaixo mostra quais as etapas desta pesquisa e quais seus objetivos.

Mineração em repositório de software Mineração em repositório de software tem a função de descrever uma extensa variedade de investigações em repositórios de software que são os elementos produzidos e armazenados ao decorrer da evolução de um software, como as informações armazenadas em sistemas de controle de versão de código fonte. Esta etapa tem o objetivo de verificar em quais contextos os programadores estão adotando o uso de expressões lambda.
